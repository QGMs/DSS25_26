# Desenvolvimento de Sistemas de Software

Sistema de gestão para uma cadeia de restaurantes de fast food, desenvolvido no âmbito da unidade curricular de Desenvolvimento de Sistemas de Software.

O projeto tem como objetivo analisar, modelar e documentar um sistema capaz de apoiar a gestão de restaurantes, pedidos, utilizadores e operações associadas ao funcionamento de uma cadeia de fast food.

## Descrição

Este projeto foca-se na fase de análise e modelação de um sistema de gestão de restaurantes. A solução pretende representar os principais processos de negócio de uma cadeia de fast food, incluindo a gestão de pedidos, gestão de utilizadores, gestão da cadeia e interação com diferentes entidades do sistema.

O trabalho inclui modelos de domínio, diagramas de casos de uso e documentação de apoio ao desenvolvimento do sistema.

## Funcionalidades modeladas

* Gestão de pedidos.
* Gestão de subpedidos.
* Gestão de utilizadores.
* Gestão da cadeia de restaurantes.
* Criação de pedidos.
* Representação das principais entidades do domínio.
* Identificação dos principais casos de uso do sistema.
* Análise das interações entre atores e funcionalidades.
